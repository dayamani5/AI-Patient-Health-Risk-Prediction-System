# AI-Powered Patient Health Risk Prediction System

A Flask + SQLite healthcare assessment project with CRUD, validation, dashboard analytics, search, risk filtering, and AI-style health risk prediction.

## Features
- Create, read, update, delete patient records
- Validate email, DOB, glucose, haemoglobin, and cholesterol
- Persistent SQLite storage
- Low / Moderate / High risk classification
- Dashboard cards and risk badges
- Search by name/email and filter by risk level

## Run
```bash
pip install -r requirements.txt
python app.py
```
Open `http://127.0.0.1:5000`.

This app is for educational assessment only and is not a medical diagnosis system.
